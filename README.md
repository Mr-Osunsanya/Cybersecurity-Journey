#  OSI Model Overview

| Layer | Name | Main Function |
|---|---|---|
| 7 | Application | User-facing network services |
| 6 | Presentation | Encryption and formatting |
| 5 | Session | Maintains communication sessions |
| 4 | Transport | Reliable data delivery |
| 3 | Network | Routing and IP addressing |
| 2 | Data Link | MAC addressing and local delivery |
| 1 | Physical | Physical transmission of data |
